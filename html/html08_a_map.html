1개의 이미지를 여러곳으로 링크걸기

        여러곳으로 링크 걸기
        1. map 태그를 이용하여 링크 정보를 기술한다.
         - 페이지 마지막 내용으로 표시를 하여야 레이아웃에 문제가 발생하지 않는다.
         - 지정할 범위를 이미지의 픽셀 단위의 x,y좌표를 시작점과 끝점을 기입한다.

        2. area 태그를 이용하여 링크 정보를 기술한다.

        shape : rect(사각형)
        coords : 좌표 (coords="x1,y1,x2,y2")

        shape : circle(원형)
        coords : 좌표 (coords="x, y, r(반지름)")

        shape : poly(다각형)
        coords : 좌표 (coords="x1,y1,x2,y2,x3,y3,x4,y4..."(꼭지점의 갯수만큼. 단, 시계방향이든 반시계방향이든 한쪽 방향으로 기술한다.))
