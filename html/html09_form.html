폼 양식 만들기

로그인폼, 회원가입, 글쓰기, 주문서 작성폼...

        text : 한줄의 문자열 입력(아이디, 이름, 주소, 글제목, 연락처...)
        name : 백엔드 서버로 데이터를 가져갈 때 필요한 변수
        value : 박스에 입력한 값을 가지는 속성
        placeholder : 안내 메시지 표시
        readonly : 읽기 전용
        disabled : 비활성화
        size : 박스크기(실제로 들어가는 글자수에 딱 떨어지지않아서 요즘은 CSS로 맞춘다.)
        minlength, maxlength : 입력할 수 있는 글자의 수 지정
        radio : 제시하는 목록 중 1개만 선택
        checkbox : 제시하는 목록 중 여러개 선택 가능
        rows : 줄 수
        cols : 한 줄당의 글자 수
        ※ <대괄호> 안에 마지막에 '/'가 있으면 끝낸다는 표시로 적는 것을 권장하지만, 없어도 기능상 문제는 없다.
